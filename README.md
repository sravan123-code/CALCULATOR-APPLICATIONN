# 🧮 Java Swing Calculator

A simple and user-friendly **Calculator Application** built using **Java Swing** for GUI and **AWT Event Handling**. This project performs basic arithmetic operations like addition, subtraction, multiplication, and division with an interactive graphical interface.

## 📌 Features

✅ Attractive GUI using Java Swing  
✅ Perform basic arithmetic operations:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

✅ Decimal number support  
✅ Clear button to reset calculations  
✅ Responsive button click actions  
✅ Real-time display updates  

## 🛠️ Technologies Used

- Java
- Swing (GUI)
- AWT
- Event Handling
- OOP Concepts
